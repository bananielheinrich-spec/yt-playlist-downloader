# ▶ YT Playlist Downloader

YouTube-Playlists als MP4 oder MP3 herunterladen – mit einfacher Browser-Oberfläche.

## ⬇ Download

**→ [Neueste Version herunterladen](../../releases/latest/download/YT-Playlist-Downloader.exe)**

Einfach `.exe` herunterladen und doppelklicken. Fertig. Kein Python, keine Installation.

## 🚀 So funktioniert es

1. `.exe` starten
2. Browser öffnet sich automatisch
3. Playlist-URL eingeben
4. Zielordner wählen
5. Herunterladen klicken

## ✨ Features

- 🎬 MP4 (Video) oder 🎵 MP3 (nur Audio)
- Fortschrittsbalken pro Video + Gesamtfortschritt
- Zeigt welches Video gerade lädt
- Ordner per Dialog auswählen
- Überspringt automatisch nicht ladbare Videos
