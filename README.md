# ▶ YT Playlist Downloader

YouTube-Playlists als MP4 oder MP3 herunterladen – mit einfacher Website-Oberfläche.

## ⚡ Voraussetzung
- [Python](https://python.org/downloads) installiert (bei Installation "Add to PATH" anhaken!)

## 🚀 Starten
1. Alle Dateien herunterladen (grüner "Code" Button → "Download ZIP")
2. ZIP entpacken
3. `START.bat` doppelklicken → Browser öffnet sich automatisch

## ✨ Features
- MP4 (Video) oder MP3 (nur Audio)
- Fortschrittsbalken pro Video + Gesamt
- Ordner per Dialog auswählen
- Überspringt automatisch nicht ladbare Videos
