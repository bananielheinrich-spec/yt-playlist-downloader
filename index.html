▶ YouTube Playlist Downloader

Lade Videos herunter – speichern auf deinem Gerät

⚙️ Setup (einmalig):

  1. Öffne Google Cloud Console
  2. Erstelle ein neues Projekt
  3. Aktiviere "YouTube Data API v3"
  4. Erstelle einen API-Key (Credentials → API Key)
  5. Kopiere deinen API-Key unten ein

📥 Videos werden direkt auf deinem Gerät gespeichert
